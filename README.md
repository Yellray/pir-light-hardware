# PIR-Light-Hardware
This repository contains the electrical schematics, PCB layout, and hardware documentation for a compact passive infrared (PIR) motion-controlled light module with ambient light threshold detection and controlled dimming ramp (fade-in/fade-out).

The system automatically controls lighting based on detected motion and ambient light conditions, ensuring the light only activates in low-light environments. It operates fully in hardware, without requiring software or network connectivity.

The module includes an I/O synchronization port that allows multiple units to be interconnected. When any connected module detects motion, all linked modules are activated simultaneously, enabling coordinated lighting across larger areas such as hallways, staircases, or multiple rooms.

Additionally, the design incorporates sensor hold and light-feedback compensation, ensuring reliable motion detection while the controlled light source is active. This prevents the PIR sensor from prematurely turning the light off due to illumination generated by the module itself, allowing the lighting to remain on continuously as long as occupancy is detected within the monitored area.
